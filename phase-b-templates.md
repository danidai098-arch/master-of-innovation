### B1-2：产品一页纸（14 字段必填）

**品牌 / 产品主张**（10-15 字）

**核心洞察**（2-3 句话：市场矛盾 + 消费者变化信号 + 现有缺口）

| 维度 | 内容 |
|------|------|
| **目标用户** | 属性 + 行为 + 情感诉求 + 原型描述（约 50 字） |
| **核心痛点 / 痒点** | 场景 + 具体困扰 |
| **RTB1·产品层** | 成分 / 配方 / 工艺差异（可验证） |
| **RTB2·体验层** | 包装 / 感官 / 使用方式差异（可感知） |
| **RTB3·商业层** | 渠道 / 定价 / 服务差异（可比较） |
| **产品 Slogan** | 8 字以内 |
| **核心传播关键词** | 词 1 · 词 2 · 词 3 |
| **内容传播钩子** | 什么内容格式最易引发 UGC |
| **竞争定位** | 对标谁 · 超越维度 · 价格带 |
| **差异化一句话** | 「我们是唯一一个 […] 的 [品类]」 |
| **品牌禁区** | 明确不做什么（人群 / 场景 / 功能 / 价格带） |
| **首购路径** | 触发点 + 渠道 |
| **复购机制** | 驱动复购的核心因素 |
| **口碑传播触发点** | 用户在什么场景会主动分享 |

## B2：MVP 测试方案

**目标**：用最低成本验证策略框架中最关键的假设。
**核心假设类型**：需求 / 解法 / 感知 / 传播 / 复购。

**MVP 形式与对应阈值**：

| MVP 类型 | 适用假设 | 做法 | 成本 / 周期 | 成功阈值 |
|---------|---------|------|------------|---------|
| 落地页测试 | 需求 | 详情页 + 少量广告 | 5000 元 / 1-2 周 | 点击率 >3% / 加购 >8% |
| 手工小批量 | 解法 | 20-50 人试用 | 低-中 / 2-4 周 | 试用复购率 >30% |
| 众筹预售 | 需求 + 支付意愿 | 摩点 / 京东众筹 | 中 / 4-6 周 | 达标完成率 |
| 达人寄测 | 感知 + 传播 | 10-20 个垂类达人 | 低 / 2-3 周 | 自发内容率 >50% |
| 快闪 / 地推 | 感知 | 目标场景体验 | 低-中 | 现场转化率 |
| 私域小群 | 复购 | 20-30 人小群免费体验 | 低 | 7 日留存 / 复购意愿 |
| 竞品盲测 | 解法 / 感知 | 用户盲测 | 低 | 选我方 >60% |

**输出格式**（推荐 1 个 MVP 方案，使用 Markdown 表格）：

### 🧪 MVP 方案：[方案名]

| 字段 | 内容 |
|------|------|
| **核心假设** | 一句话表述要验证的命题（含可量化结果） |
| **MVP 类型** | 从上表选取（可组合） |
| **测试周期** | X 周 |
| **所需资源** | 含估算金额、人员、物料 |
| **成功标准** | Go / 观察 / No-Go 三档阈值 |
| **风险提示** | 测不到什么 + 干扰因素 |
