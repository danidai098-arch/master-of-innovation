## 样例品类：功能性饮料 · 方向「4 小时清醒配方」

### B1-2 一页纸节选

**品牌主张**：4 小时清醒，不熬夜

**核心洞察**：办公室白领既想提神又怕影响睡眠；现有能量饮料全部粗暴大剂量；茶氨酸已被茶文化教育，但没有饮料产品把"短效+助眠"做成核心卖点。

| 维度 | 内容 |
|------|------|
| **目标用户** | 25-35 岁加班白领；下午 2-4 点需要专注力；晚上希望 11 点能入睡；不愿喝咖啡因炸弹 |
| **核心痛点** | 「下午想提神又怕晚上睡不着」 |
| **RTB1·产品层** | 短效咖啡因 80mg + 茶氨酸 200mg 复配 |
| **RTB2·体验层** | 罐身正面印「11pm 前清醒」失效时间标签 |
| **RTB3·商业层** | 写字楼无人货架渠道；定价 12 元/罐 |
| **产品 Slogan** | 4 小时清醒 |
| **核心关键词** | 短效 · 不熬夜 · 茶氨酸 |
| **UGC 钩子** | 晒「11pm 准时睡着」对比图 |
| **竞争定位** | 对标红牛/魔爪；超越=可控时长；价格带 10-15 元 |
| **差异化一句话** | 我们是唯一一个**让你下午喝、晚上睡**的能量饮料 |

**学习要点**：每个字段都有**可验证/可感知/可比较**的具体内容，禁止空洞营销词。

### B2 MVP 方案节选

### 🧪 MVP 方案：写字楼无人货架快闪 8 周

| 字段 | 内容 |
|------|------|
| **核心假设** | 写字楼白领愿意为「短效不影响睡眠」概念支付 12 元/罐溢价，且周复购率 >25% |
| **MVP 类型** | 快闪/地推 + 私域小群（验证感知 + 复购） |
| **测试周期** | 8 周（前 4 周铺货，后 4 周观察复购） |
| **所需资源** | 30,000 元（小批量代工 5000 罐 + 5 栋写字楼货架租金 + 群运营） |
| **成功标准** | Go：周复购 >25% + 自发分享率 >15%；观察：复购 15-25%；No-Go：<15% |
| **风险提示** | 写字楼人群偏理性，对"失效时间标签"可能误读为产品缺陷；需配店员讲解 |

**执行步骤**：
1. 第 1-2 周：小批量打样 5000 罐 + 标签设计验收
2. 第 3-4 周：5 栋写字楼货架铺货，配试饮台 + 标签讲解
3. 第 5-8 周：每周复盘销售数据 + 群内问卷追踪睡眠反馈
